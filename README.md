# ООО «Тюменьавтотранс»

Статический сайт транспортного предприятия ООО «Тюменьавтотранс» для публикации на GitHub Pages.

## Публикация на GitHub Pages

1. Создайте новый репозиторий на GitHub, например `tyumenavtotrans-site`.
2. Загрузите в репозиторий файлы из этой папки: `index.html`, `styles.css`, `script.js`, `assets/`, `.nojekyll`.
3. Откройте `Settings` -> `Pages`.
4. В блоке `Build and deployment` выберите `Deploy from a branch`.
5. Выберите ветку `main` и папку `/root`.
6. Сохраните настройки и дождитесь публикации.

После публикации сайт будет доступен по адресу вида:

```text
https://<ваш-логин>.github.io/tyumenavtotrans-site/
```
